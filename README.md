# Dotfiles

## Installation

(Assumes a [Debian based distribution](https://en.wikipedia.org/wiki/List_of_Linux_distributions#Debian-based).)

```bash
curl -o- https://raw.githubusercontent.com/cooperpellaton/dotfiles/master/setup.sh | sudo bash
```
